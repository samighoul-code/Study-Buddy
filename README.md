# منصة رياضيات المرحلة المتوسطة 🎓

تطبيق ويب تعليمي للصفوف 7-9 يجمع بين تعلم الرياضيات وإدارة المهام.

## المميزات

- **لوحة التحكم**: ترحيب بالطالب، نقاط، مستوى، أوسمة
- **الرياضيات**: 6 مواضيع، 48 سؤالاً، 3 مستويات صعوبة
- **إدارة المهام**: إضافة وتتبع الواجبات والمهام الدراسية
- **نظام نقاط وأوسمة**: تحفيز الطلاب على الاستمرار
- **دعم كامل للغة العربية** واتجاه RTL

## المواضيع الرياضية

| الموضوع | الصف | الأسئلة |
|---------|------|---------|
| الدوال التربيعية | الثامن | 8 أسئلة |
| المعادلات التربيعية | الثامن | 8 أسئلة |
| المعادلات الخطية | السابع | 8 أسئلة |
| أنظمة المعادلات | التاسع | 8 أسئلة |
| التحويلات الهندسية | السابع | 8 أسئلة |
| المتباينات التربيعية | التاسع | 8 أسئلة |

## كيفية التشغيل على Replit

1. افتح المشروع في Replit
2. اضغط **Run** لتشغيل التطبيق
3. افتح الرابط في نافذة المعاينة
